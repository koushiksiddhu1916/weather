# WeatherPro - Real-Time Weather Application

A modern, responsive weather web application with a beautiful glassmorphism UI.

## Live Demo
[Click here to view live](https://9kwds3.csb.app/)  
(Replace with your actual deployed link)

## Features
- Search weather by city name
- Get current location weather with one click
- Real-time data from OpenWeatherMap API
- Temperature in Celsius and Fahrenheit (toggle)
- Displays: Feels like, Humidity, Wind Speed, Pressure, Visibility
- Smooth animations and particle background
- Fully responsive design
- Saves last search and unit preference
- Keyboard shortcuts supported

## How to Run Locally
1. Clone or download this repository
2. Open `index.html` in any browser

No build tools or server required!

## API Configuration
1. Get free API key from [OpenWeatherMap](https://openweathermap.org/api)
2. Open `script.js`
3. Replace the API key:

```js
const CONFIG = {
    API_KEY: 'your_actual_api_key_here',
    // ...
};
```
